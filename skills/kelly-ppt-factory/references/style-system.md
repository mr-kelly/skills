# PPTX Style Kit

A style kit is a reusable presentation contract. It should be created before generating bulk decks.

Required pieces:

- Palette: 3-5 colors with one stable navigation color, one neutral background, and one or two accents.
- Fonts: heading, body, and optional CJK font.
- Visual rules: illustration/photo style, crop rules, asset quality floor, forbidden drift.
- Layout rules: recurring slide families and density limits.
- Component library: repeated parts like title rails, metric callouts, proof blocks, speaker-note strips, page numbers, and QA badges.

For any project, define the audience, deck use case, visual evidence style, copy density, and QA risks before generating at scale.
