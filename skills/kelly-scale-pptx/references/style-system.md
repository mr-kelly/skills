# PPTX Style System

A style system is a reusable courseware contract. It should be created before generating bulk decks.

Required pieces:

- Palette: 3-5 colors with one dominant teaching color, one stable navigation color, and one accent.
- Fonts: heading, body, and optional Chinese font.
- Visual rules: illustration/photo style, crop rules, asset quality floor, forbidden drift.
- Layout rules: recurring slide families and density limits.
- Component library: repeated parts like title bands, prompt panels, vocabulary chips, teacher-note strips, page numbers, and QA badges.

For children's Chinese courseware, prefer warm, inspectable visuals and large Chinese text. Keep pinyin helpful but secondary. English is a support layer, not the main visual object unless the deck explicitly teaches bilingual comparison.
