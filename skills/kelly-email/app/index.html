Configuration

Busabase provider storage:

busabase:drive/config/config.json
busabase:vault/kelly-email

Local provider files:

.agents/skills/kelly-email/config.local.json
.agents/skills/kelly-email/.env.local

Kelly Email reads configuration and review state through the selected data provider. Local mode uses app files; Busabase mode lazily initializes a workspace Folder, structured Base, and app-state Drive before review actions are enabled.

JSON stores accounts, aliases, identities, user profile, brands, official URLs, knowledge sources, style, CTA URLs, and risk keywords. Busabase mode stores secret values in Busabase Vault and references them by vault_ref.
